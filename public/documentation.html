Movie API Documentation

This API allows users to retrieve movie information and manage user accounts, including registering users, updating profiles, and managing favorite movies.

Get All Movies

Description: Returns a list of all movies in the database.

  • URL: /movies
  • Method: GET
  • URL Parameters: None
  • Request Body: None
  • Response Format: JSON array of movie objects
[
  {
    "Title": "Inception",
    "Description": "A mind-bending thriller...",
    "Genre": { "Name": "Sci-Fi", "Description": "Science fiction movies" },
    "Director": { "Name": "Christopher Nolan", "Bio": "...", "BirthYear": 1970 },
    "ImageURL": "http://example.com/inception.jpg",
    "Featured": true
  }
]

Get Movie by Title

Description: Returns detailed data about a single movie by title.

  • URL: /movies/:title
  • Method: GET
  • URL Parameters:
    • title – The title of the movie
  • Request Body: None
  • Response Format: JSON object
{
  "Title": "Inception",
  "Description": "A mind-bending thriller...",
  "Genre": { "Name": "Sci-Fi", "Description": "Science fiction movies" },
  "Director": { "Name": "Christopher Nolan", "Bio": "...", "BirthYear": 1970 },
  "ImageURL": "http://example.com/inception.jpg",
  "Featured": true
}

Get Genre by Name

Description: Returns data about a genre by name.

  • URL: /movies/genre/:genreName
  • Method: GET
  • URL Parameters:
    • genreName – Name of the genre
  • Request Body: None
  • Response Format: JSON object
{
  "Name": "Thriller",
  "Description": "Thrilling and suspenseful movies"
}

Get Director by Name

Description: Returns data about a director by name.

  • URL: /movies/director/:directorName
  • Method: GET
  • URL Parameters:
    • directorName – Name of the director
  • Request Body: None
  • Response Format: JSON object
{
  "Name": "Christopher Nolan",
  "Bio": "British-American film director...",
  "BirthYear": 1970,
  "DeathYear": null
}

Register New User

Description: Allows a new user to register.

  • URL: /users
  • Method: POST
  • URL Parameters: None
  • Request Body: JSON object
  • Response Format: JSON object
Request Body Example:
{
  "name": "john_doe",
  "favoriteMovies": []
}
  
Response Example:
{
  "id": "uuid-value",
  "name": "john_doe",
  "favoriteMovies": []
}

Update User Name

Description: Updates an existing user's name.

  • URL: /users/:id
  • Method: PUT
  • URL Parameters:
    • id – User ID
  • Request Body: JSON object
  • Response Format: JSON object
Request Body Example:
{
  "name": "new_username"
}

Add Favorite Movie

Description: Adds a movie to a user's list of favorite movies.

  • URL: /users/:id/:movieTitle
  • Method: POST
  • URL Parameters:
    • id – User ID
    • movieTitle – Movie title
  • Request Body: None
  • Response Format: Text message
Example Request:
POST /users/12345/Inception

Response Example:
"Inception has been added to user 12345's array"

Remove Favorite Movie

Description: Removes a movie from a user's favorites.

  • URL: /users/:id/:movieTitle
  • Method: DELETE
  • URL Parameters:
    • id – User ID
    • movieTitle – Movie title
  • Request Body: None
  • Response Format: Text message
Example Request:
DELETE /users/12345/Inception

Response Example:
"Inception has been removed from user 12345's array"

Deregister User

Description: Deletes an existing user account.

  • URL: /users/:id
  • Method: DELETE
  • URL Parameters:
    • id – User ID
  • Request Body: None
  • Response Format: Text message
Example Request:
DELETE /users/12345

Response Example:
"user 12345 has been deleted"
