Movie API Documentation

This API provides access to movie and user data using MongoDB and Mongoose. Users can register, update their profiles, manage favorite movies, and retrieve information about movies, genres, directors, and actors.

Get All Movies

Description: Returns a list of all movies.

  • URL: /movies
  • Method: GET
  • URL Parameters: None
  • Request Body: None
  • Response Format: JSON array

Get Movie by Title

Description: Returns detailed data about a movie.

  • URL: /movies/:title
  • Method: GET
  • URL Parameters: title
  • Request Body: None
  • Response Format: JSON object
{
  "title": "The Matrix",
  "description": "A hacker discovers reality is a simulation.",
  "genre": {
    "name": "Sci-Fi",
    "description": "Science fiction films"
  },
  "director": {
    "name": "Wachowski Sisters"
  },
  "actors": ["Keanu Reeves", "Laurence Fishburne"],
  "releaseYear": 1999
}

Get Genre by Name

Description: Returns genre information.

  • URL: /movies/genre/:genreName
  • Method: GET
  • URL Parameters: genreName
  • Request Body: None
  • Response Format: JSON object

Get Director by Name

Description: Returns director information.

  • URL: /movies/director/:directorName
  • Method: GET
  • URL Parameters: directorName
  • Request Body: None
  • Response Format: JSON object

Get Movies by Actor

Description: Returns all movies featuring a given actor.

  • URL: /movies/actors/:actorName
  • Method: GET
  • URL Parameters: actorName
  • Request Body: None
  • Response Format: JSON object
{
  "actor": "Keanu Reeves",
  "movies": ["The Matrix", "John Wick"]
}

Register New User

Description: Registers a new user.

  • URL: /users
  • Method: POST
  • Request Body: JSON object
  • Response Format: JSON object
{
  "userName": "johnDoe",
  "password": "password123",
  "email": "john@example.com",
  "birthday": "1990-01-01"
}

Get All Users

  • URL: /users
  • Method: GET
  • Response Format: JSON array

Get User by Username

  • URL: /users/:userName
  • Method: GET
  • URL Parameters: userName
  • Response Format: JSON object

Update User Profile

  • URL: /users/:userName
  • Method: PUT
  • URL Parameters: userName
  • Request Body: JSON object
  • Response Format: JSON object

Add Favorite Movie

  • URL: /users/:userName/movies/:MovieID
  • Method: POST
  • Response Format: JSON object

Remove Favorite Movie

  • URL: /users/:userName/movies/:movieId
  • Method: DELETE
  • Response Format: JSON object

Delete User

  • URL: /users/:userName
  • Method: DELETE
  • Response Format: Text
