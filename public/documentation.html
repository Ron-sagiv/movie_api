Movie API Documentation

This API provides access to movie data and user data. Users can register, update their profiles, manage favorite movies, and retrieve information about movies, genres, directors, and actors.

Get All Movies

Description: Returns a list of all movies.

  • URL: /movies
  • Method: GET
  • URL Parameters: None
  • Request Body: None
  • Response Format: JSON array of movie objects

Get Movie by Title

Description: Returns detailed data about a movie by title.

  • URL: /movies/:title
  • Method: GET
  • URL Parameters: title
  • Request Body: None
  • Response Format: JSON object
{
  "title": "The Matrix",
  "director": { "name": "Wachowski sisters" },
  "released": 1999,
  "genre": { "name": "sci-fi" },
  "actors": ["Keanu Reeves", "Laurence Fishburne"]
}

Get Genre by Name

Description: Returns genre data by name.

  • URL: /movies/genre/:genreName
  • Method: GET
  • URL Parameters: genreName
  • Request Body: None
  • Response Format: JSON object
{
  "name": "crime"
}

Get Director by Name

Description: Returns director data by name.

  • URL: /movies/director/:directorName
  • Method: GET
  • URL Parameters: directorName
  • Request Body: None
  • Response Format: JSON object
{
  "name": "Christopher Nolan"
}

Register New User

Description: Registers a new user.

  • URL: /users
  • Method: POST
  • URL Parameters: None
  • Request Body: JSON object
  • Response Format: JSON object
Request Body Example:
{
  "name": "John Doe",
  "email": "john@example.com"
}

Update User Name

Description: Updates a user's name.

  • URL: /users/:id
  • Method: PUT
  • URL Parameters: id
  • Request Body: JSON object
  • Response Format: JSON object
{
  "name": "New Name"
}

Add Favorite Movie

Description: Adds a movie to a user's favorites.

  • URL: /users/:id/:movieTitle
  • Method: POST
  • URL Parameters: id, movieTitle
  • Request Body: None
  • Response Format: Text
Example Response:
"The Matrix added to user 1"

Remove Favorite Movie

Description: Removes a movie from favorites.

  • URL: /users/:id/:movieTitle
  • Method: DELETE
  • URL Parameters: id, movieTitle
  • Request Body: None
  • Response Format: Text

Deregister User

Description: Deletes a user account.

  • URL: /users/:id
  • Method: DELETE
  • URL Parameters: id
  • Request Body: None
  • Response Format: Text
